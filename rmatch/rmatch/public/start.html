Quick Start

Last updated: Nov 10th, 2016

Request

Request URL
http://rmatch.herokuapp.com/instance/query
Parameters
Name Type Description Default
apikey string This is a required parameter, please contact the developer for your key
instance intiger This ID will determine with which user set to comare the set answers
q intiger This is an intiger with the question ID
answin intiger This will be the user's answer to the question with ID:q
ansimp intiger This will be the importance to user's answer to the question with ID:q
Example
require 'net/http' require 'uri'

uri = URI.parse( "http://rmatch.herokuapp.com/instance/query" ) 
params = {'apikey':'rmatch_key', 'instance':5, 'q':3, 'answin':2, 'ansimp':5}

http = Net::HTTP.new(uri.host, uri.port) 
request = Net::HTTP::Get.new(uri.path) 
request.set_form_data( params )

# instantiate a new Request object
request = Net::HTTP::Get.new( uri.path+ '?' + request.body ) 

response = http.request(request)
puts response.body
